<br/>

> **Full-stack enterprise procurement management platform** with multi-role access and end-to-end procurement lifecycle support

- 👥 **Role-Based Access Control** — Dedicated dashboards for Admin, Procurement Officer, Manager, and Vendor with JWT Authentication
- 📋 **Procurement Modules** — Vendor onboarding, RFQ management, bid submission, bid evaluation, approval workflows, and Purchase Order generation
- 🤖 **Smart Bidding** — Automated bid comparison and Lowest Bidder (L1) identification for streamlined procurement decisions
- 🔍 **Audit Trail** — Full activity logging for transparency, compliance, and procurement lifecycle tracking
- 🎨 **Modern UI** — Responsive Glassmorphism design built with React.js and Tailwind CSS
